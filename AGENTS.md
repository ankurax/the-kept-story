# The Kept Story

The Kept Story is a private, cinematic storytelling website built with HTML, CSS and vanilla JavaScript.

## Project Goals

- Mobile-first
- Fast loading
- Elegant animations
- Emotional storytelling
- Minimal dependencies
- No frameworks

## Design Language

- Warm and minimal
- Rounded cards
- Soft shadows
- Smooth transitions
- Comfortable spacing
- Consistent typography

## Code Style

- Use semantic HTML.
- Keep CSS organized by section.
- Use descriptive class names.
- Avoid inline styles.
- Avoid duplicate CSS.
- Prefer reusable components.
- Keep JavaScript modular.

## Existing Sections

- Landing Page
- Firsts
- Time Did Tell
- Remaining Categories

## Before Editing

Always preserve:

- responsive layout
- smooth scrolling
- animations
- accessibility
- existing visual style

Do not rewrite unrelated code.

Only change what is necessary.

If creating a new section, match the existing design language.
